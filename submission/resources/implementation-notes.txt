Feature 1 - Loop-rendered dynamic content:
- Added a new "Vendor Services" section in index.html with an empty container: id="services-list".
- Implemented JavaScript in js/script.js using an array of objects and forEach().
- Each object includes a required "name" property and a "price" property.
- Service cards are generated dynamically with createElement(), innerHTML, and appendChild().

Feature 2 - Add & Remove items:
- Added wishlist UI in index.html with required elements: input id="item", Add button, and ul id="wishlist".
- Used createElement() to create li, span, and Remove button for each new item.
- Used appendChild() to attach each element to the DOM.
- Used remove() so each item's Remove button deletes that specific item.

Feature 3 - Form validation:
- Added a contact form with name and email inputs plus a feedback paragraph.
- Used event.preventDefault() on form submit to handle validation in JavaScript.
- Read user input using .value from name and email fields.
- Displayed dynamic feedback with textContent:
	- Success: shown in a custom popup modal as "Thank you [Name]. We'll contact you shortly."
	- Invalid email: "Email is invalid."

Feature 4 - localStorage persistence:
- Added a Dark Mode toggle button and saved theme preference using localStorage.setItem("theme", "dark"/"light").
- On page load, the site reads localStorage.getItem("theme") and applies the saved theme automatically.
- Added extra persistence by saving last contact name and email while typing, then restoring them on reload.
